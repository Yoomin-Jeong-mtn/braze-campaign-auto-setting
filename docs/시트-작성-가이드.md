# 푸시 캠페인 세팅 시트 작성 가이드

---

## 세팅이 되는 대상

**발송일이 내일**인 캠페인 중 상태가 **작성 완료**인 행을 자동으로 세팅합니다.

> **금요일**: 토요일 + 일요일 + 월요일 캠페인을 함께 처리합니다.

자동 실행 시각: **평일 낮 12:30 (KST)**

---

## 열 설명

### 직접 입력하는 열

| 열 | 항목 | 입력 형식 | 예시 |
|---|---|---|---|
| A | 상태 | 아래 [상태 안내] 참고 | `작성 완료` |
| B | 발송 시각 | 12시간제, `HH:MM AM/PM` | `11:00 AM` / `02:30 PM` |
| C | 발송일 | `YYYY/MM/DD` | `2026/06/13` |
| D | 방송번호 | 숫자 | `26942` |
| E | 캠페인명 | Braze에 표시될 캠페인 이름 | `AH_SAL_LVC_APP_PUSH_SCH_26942` |
| F | 제목 | 푸시 알림 제목 (광고 문구 자동 조립됨) | `오늘만! 라이브 특가` |
| G | 본문 | 푸시 알림 본문 (광고/수신거부 문구 자동 조립됨) | `지금 바로 확인하세요` |
| H | 세그먼트 | 세그먼트명 (별도 탭의 목록과 일치해야 함) | `패션` / `여행` / `라이프` |
| I | Android 이미지 | 이미지 URL | `https://...` |
| J | iOS 이미지 | 이미지 URL | `https://...` |

### 자동으로 채워지는 열 (입력 불필요)

| 열 | 항목 | 내용 |
|---|---|---|
| K | Braze URL | 세팅 완료 후 해당 캠페인 Braze 링크 |
| L | 에러 원인 | 세팅 실패 시 원인 메시지 |
| M | 업데이트 일자 | 마지막으로 상태가 바뀐 시각 |

---

## 상태 안내

| 상태 | 누가 설정 | 의미 |
|---|---|---|
| `작성 중` | 작성자 | 아직 입력 중, 자동 세팅 대상 아님 |
| `작성 완료` | 작성자 | 입력 완료, 다음 실행 시 자동 세팅 대상 |
| `세팅 완료` | 시스템 | Braze Draft 생성 완료 (K열에 링크 확인) |
| `세팅 실패` | 시스템 | 세팅 중 오류 발생 (L열에 원인 확인) |

> **재시도**: 세팅 실패한 행은 상태를 다시 `작성 완료`로 변경하면 다음 실행 때 재시도합니다.

---

## 자동 조립되는 내용

F열(제목)과 G열(본문)에 입력한 내용에 아래 문구가 자동으로 붙습니다. 따로 입력하지 않아도 됩니다.

- **제목**: `{{content_blocks.${push_setting}}} + 입력값`
- **본문**: `(광고) + 입력값 + \n*수신거부:설정>알림설정`
- **딥링크**: 방송번호(D열)로 라이브 방송 URL 자동 생성

---

## 주의사항

- **세그먼트명**은 반드시 `세그먼트별 템플릿 캠페인 id` 탭에 등록된 이름과 정확히 일치해야 합니다. 오타가 있으면 `세팅 실패` 처리됩니다.
- **발송일 기준**: 평일에는 내일 발송 캠페인만, 금요일에는 토요일 + 일요일 + 월요일 발송 캠페인까지 처리합니다. 월요일 캠페인은 전주 금요일까지 작성 완료 상태로 입력해주세요.
- **최종 확인**: 세팅 완료 후 K열 링크로 Braze에서 내용을 한 번 검토하세요. 자동 세팅이므로 발송 전 확인을 권장합니다.
